# AI/ML 检索策略

本项目跟踪 AI、机器学习、深度学习、大模型和具身智能，不只关注单一模型或单一会议。检索词按以下方向组织：

- 基础 AI/ML：artificial intelligence、machine learning、deep learning、neural network
- 大模型：large language model、LLM、foundation model、generative AI、Transformer
- 多模态与生成模型：vision-language model、multimodal、diffusion model、text-to-image、text-to-video
- 智能体与强化学习：AI agent、multi-agent、reinforcement learning、planning、tool use
- 具身智能：embodied AI、robot learning、world model、humanoid、manipulation、navigation
- 安全与对齐：AI safety、AI alignment、RLHF、hallucination、jailbreak、red teaming
- 系统与效率：distributed training、efficient inference、model compression、quantization、serving

ranking 中的强相关词决定哪些论文进入日报；support terms 用于提升排序；exclude terms 用于过滤常见歧义。
